# Getting started

1. To get a local copy up and running, follow these simple steps.

`git clone https://github.com/faissal17/AlloMedia-RESTAPI.git`

2. navigate the project

`cd AlloMedia-RESTAPI`

3.  Install dependencies using npm

`npm install`
